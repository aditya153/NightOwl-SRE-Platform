Triage Agent

The first responder. Assesses incoming alerts, determines severity, sets priority, and routes to appropriate agents.

Severity Assessment Priority Setting Routing
📄

Log Analyst

Parses logs from Loki, identifies error patterns, and performs root cause analysis using log data.

Log Parsing Pattern Detection RCA
📈

Metric Correlator

Queries Prometheus metrics, correlates time-series data, and detects anomalies across services.

PromQL Queries Correlation Anomaly Detection
🔧

Fixer Agent

Executes remediation actions — restart deployments, rollback releases, scale resources.

Restart Rollback Scale
🛡️

Security Auditor

Performs SAST scans, CVE detection, secret scanning, and vulnerability assessment.

SAST CVE Scan Secret Detection
✅

Compliance Checker

Validates infrastructure against SOC2, GDPR, and custom policies. Detects configuration drift.

Policy Check SOC2/GDPR Drift Detection
